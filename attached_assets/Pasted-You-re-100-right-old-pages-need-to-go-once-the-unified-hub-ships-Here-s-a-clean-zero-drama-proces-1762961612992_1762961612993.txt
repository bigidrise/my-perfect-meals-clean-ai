## Phase C — Remove + sanity checks

1. Delete deprecated files:

   * Example:

     * `client/src/pages/DiabetesSupportPage.tsx`
     * (plus any other now-unused diabetic/blood sugar pages)
2. Remove/adjust exports from any barrels:

   * `client/src/pages/index.ts`
3. Typecheck + build:

   * `npm run typecheck` (or `tsc --noEmit`)
   * `npm run build`
4. Prune unused deps (optional but smart):

   * Add dev tools to spot stragglers:

     ```json
     // package.json
     "devDependencies": {
       "knip": "^5.0.0",
       "ts-prune": "^0.10.3"
     },
     "scripts": {
       "lint:unused": "knip",
       "lint:dead": "ts-prune"
     }
     ```
   * Run: `npm run lint:unused` and `npm run lint:dead`

## Phase D — Commit discipline

* One PR/commit titled: `feat(diabetic-hub): add unified page; remove legacy blood sugar pages`
* No mixed changes. No sneaky refactors.
