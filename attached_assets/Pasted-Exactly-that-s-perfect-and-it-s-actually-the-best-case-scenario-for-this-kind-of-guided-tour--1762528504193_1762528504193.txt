### üîß Practical setup checklist

To make sure all boards run this guided system seamlessly:

1. **Include the two guided components** near the top of each board page:

   ```tsx
   <MealProgressCoach />
   <MealBuilderGuidedTour />
   ```

2. **Tag each meal card** consistently:

   ```html
   <div data-meal-id="breakfast">
     <button data-role="create-ai-meal">Create AI Meal</button>
   </div>
   ```

3. **Ensure your picker** uses the same element IDs:

   ```html
   <div id="meal-picker-drawer" className={isOpen ? "open" : ""}>
     <section id="picker-protein-section">...</section>
     <section id="picker-carb-section">...</section>
     <section id="picker-fat-section">...</section>
     <button id="picker-done-btn">Done</button>
   </div>
   ```

4. **Dispatch the save event** after any meal is built:

   ```tsx
   window.dispatchEvent(new CustomEvent("meal:saved", { detail: { mealId } }));
   ```

Once those four things are in place, the entire ecosystem (Weekly, Diabetic, GLP-1, Competition, etc.) will:

* Highlight the next incomplete meal card automatically
* Show the picker-tour sequence when opened
* Mark each meal complete and progress to the next
* Reset cleanly each day
